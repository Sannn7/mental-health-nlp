# Mental Health Signal Extraction from Reddit Posts
### An End-to-End NLP Pipeline

> An end-to-end NLP pipeline that ingests Reddit mental health posts, extracts structured insight from unstructured text, and 
> surfaces a searchable, summarized, and analyzed view of what people are actually experiencing.

---

## Problem Statement

People on Reddit's mental health subreddits share raw, unstructured experiences
every day. This pipeline ingests those posts and answers:

> *"What are people actually experiencing, and how can we surface structured
> insight from this unstructured text?"*

The output is a full intelligence engine: posts are cleaned, classified, entity-
extracted, topic-clustered, summarized, and made semantically searchable.

## Quickstart

### 1. Install dependencies

```bash
pip install -r requirements.txt
python -m spacy download en_core_web_sm
```

### 2. Download the Kaggle dataset

Set up your Kaggle API key (`~/.kaggle/kaggle.json`) then:

```bash
python run_pipeline.py --module 1
```

Or manually download from Kaggle and place the CSV at:
`data/raw/reddit_mental_health.csv`

### 3. Run the full pipeline

```bash
python run_pipeline.py --all
```

Or module by module (recommended for first run):

```bash
python run_pipeline.py --module 1   # clean data
python run_pipeline.py --module 2   # classical NLP
python run_pipeline.py --module 3   # transformers (GPU recommended)
python run_pipeline.py --module 4   # summarization
python run_pipeline.py --module 5   # build search index
```

### 4. Search interactively

```bash
python -m src.search.semantic_search --query "sleep problems and medication"
python -m src.search.semantic_search --query "feeling hopeless and isolated"
```
